# SOC Analyst Lab Portfolio

## Overview

This repository documents hands-on cybersecurity labs focused on SOC analyst skills, Windows log analysis, Splunk, phishing investigation, PowerShell automation, Python alert triage, MITRE ATT&CK mapping, vulnerability management, and incident response documentation.

The goal of this portfolio is to demonstrate practical skills for entry-level SOC Analyst, Cybersecurity Analyst, and Security Operations roles.

* SOC alert triage

* Incident response documentation
